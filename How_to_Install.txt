
first install the colored module for python3

$ pip install colored

Then if you are using Windows, just simply have the Programs folder in thesame directory as the python file and everything should work

On Linux however you must first install youtube-dl and ffmpeg.

# Installing youtube-dl on Ubuntu Linux via PPA commands

$ sudo apt update && sudo apt upgrade
$ sudo add-apt-repository ppa:nilarimogard/webupd8
$ sudo apt-get update
$ sudo apt-get install youtube-dl

# installing youtube-dl on Kali Linux

$ sudo apt install youtube-dl

# Installing ffmpeg on Ubuntu Linux via PPA commands

$ apt-get install ffmpeg
